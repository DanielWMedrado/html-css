Negrito / Destaque

Nessa frase temos uma plavra em negrito usando a tag <b> não é semantica pois ela so tem forma mas a tag <strong> tem semantica pois a palavra significa forte

Italico / Ênfase

Nesta frase, eu usarei a tag <i> para italico e claramente a tag não é semantica

Nesta frase, temos um termo em ênfase usando a tag <em> semantica

Texto Marcado

Podemos marcar um texto usando a tag <mark>

Tag <style>

O tag style adiciona um parametro em um uma tag colocando a tag style dentro de uma outra tag
<mark style="background-color: "green"> mudando a cor da marcação <mark/>

Podemos tambem definir uma cor para todas as tags usando a tag style de uma forma diferente

<style>
mark {
background-color: "aquamarine"
}
<style/>

Texto Grande e pequeno

Estamos criando um <big>texto grande<big> e um <small>texto pequeno<small> nesse parágrafo.

Texto deletado

Podemos marcar um <del>texto como excluído<del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marrcar um <ins>texto como inserido<ins> para dar uma ênfase e indicar que ele foi adicionado depois.
