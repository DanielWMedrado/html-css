Favicon (Icone de favorito)

Ao manipular imagens podemos
definir um favicon para o nosso
site que é o icone da guia do seu site

Uma maneira de encontrar icones gratuitos para o favicon é pelo site IconArchive

Você tambem pode criar o seu atravez do favicon.cc

Caso queira converter uma imagem, texto etc... Em um icone para isso use o site favicon.io

Para inserir a favicon no site é só digitar link e na aba de sugestões da ide selecionar o link:favicon

OBS: Não é obrigatorio mas é recomendado renomear o icone para favicon
