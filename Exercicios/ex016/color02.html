Testando Gradiente

Tipos de Gradiente

para inserir um gradiente iremos usar o background-image inplementando:

Linear-gradiente: nele iremos definir gradinte segindo uma linha podendo usar como criterio angulo ou direção como no Exemplo:

background-image: linear-gradient(90deg(90 graus), #3A5DE6, #8196EA);

Podemos definir quantas cores for desejada

Radial-gradient: Esse vai definir o gradiente a partir de um formato como circle

background-image: radial-gradient(circle, #75CBE6, #BDDFEA);

Não exagere na quantidade de cores
